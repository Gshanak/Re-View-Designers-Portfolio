# Re-View Designers Agency Portfolio 🚀

A high-performance, professional agency portfolio built for **Re-View Designers**, focusing on digitizing local retail businesses in Srikakulam, Andhra Pradesh. This project follows a "Stock UI" aesthetic—prioritizing speed, clarity, and mobile-first responsiveness.

## 🌟 Key Features

- **Performance-First Design:** Minimalist Stock UI inspired by high-end Dribbble showcases.
- **WhatsApp Integration:** Direct "One-Tap" lead generation for local business owners.
- **QR Pill Navigation:** A custom floating "Scan Me" pill that links directly to our Google Business Profile.
- **Hybrid Data Management:** Uses a predefined project array for instant loading with a hidden LocalStorage-based Admin Dashboard for rapid prototyping.
- **Vibe Coded Architecture:** Optimized for fast deployment on Vercel without the overhead of a heavy backend.

## 🛠 Tech Stack

- **Frontend:** HTML5, Tailwind CSS, Vanilla JavaScript
- **Storage:** LocalStorage (for client-side management)
- **Deployment:** Vercel
- **Development Environment:** Linux Mint 22
